```markdown
# OpenMatrix: 一个让 AI 成为项目经理的开源框架

## 背景
在 AI 辅助开发的时代，我们面临一个挑战：如何让 AI 完成复杂的多步骤任务，同时保持代码质量和可追踪性？

## 什么是 OpenMatrix？
OpenMatrix 是一个 AI 原生的任务编排框架，通过结构化的 Phase → Task → Subagent 层级，让 AI 驱动的复杂项目变得可控、可追踪、高质量。

## 核心特性

### 1. 三级质量配置
- **strict**: TDD + 80%覆盖率 + 严格Lint + 安全扫描 + AI验收
- **balanced**: 60%覆盖率 + AI验收
- **fast**: 无质量门禁，最快速度

### 2. 六道质量门禁
1. 编译检查 (必须通过)
2. 测试运行 (必须通过)
3. 覆盖率检查
4. Lint 检查
5. 安全扫描
6. 验收标准

### 3. Meeting 机制
遇到阻塞时创建记录，但不停止执行，最后统一处理。
```

````markdown
# OpenMatrix v0.1.1 Released! 🚀

We're excited to announce the first public release of **OpenMatrix** - an AI-native task orchestration framework that combines TDD, strict quality gates, and fully automated execution.

## Key Features

| Feature | Description |
|---------|-------------|
| **TDD Built-in** | strict mode enforces Test-Driven Development |
| **Quality Gates** | 6 quality gates: build, test, coverage, lint, security, acceptance |
| **Meeting Mechanism** | Blockers don't stop execution - handle them later |
| **AI Acceptance** | Reviewer Agent validates all deliverables |
| **100% Automation** | auto mode runs without any human intervention |

## Quick Start

```bash
# Install
````
