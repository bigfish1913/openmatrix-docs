## еј•иЁҖпјҡAI зј–з Ғзҡ„дҝЎд»»еҚұжңә

AI зј–з Ғе·Ҙе…·е·Із»ҸйқһеёёејәеӨ§пјҢдҪҶз”ЁжҲ·д»Қз„¶дёҚж•ўе®Ңе…ЁдҝЎд»»гҖӮдёәд»Җд№Ҳпјҹ

```

з¬¬дәҢеұӮпјҡAI еҜ№иҜқзј–зЁӢпјҲClaude CodeпјүвҶ’ и§ЈеҶігҖҢеӨҚжқӮд»»еҠЎгҖҚзҡ„й—®йўҳ
з¬¬дёүеұӮпјҡжөҒзЁӢзј–жҺ’ + иҙЁйҮҸдҝқйҡңпјҲOpenMatrixпјүвҶ’ и§ЈеҶігҖҢеҸҜдҝЎдәӨд»ҳгҖҚзҡ„й—®йўҳ
```

OpenMatrix е®ҡдҪҚеңЁз¬¬дёүеұӮвҖ”вҖ”дёҚжҳҜи®© AI еҶҷеҫ—жӣҙеҘҪпјҢиҖҢжҳҜи®© AI зҡ„дә§еҮә**еҸҜдҝЎгҖҒеҸҜдәӨд»ҳ**гҖӮ

жң¬ж–Үе°Ҷж·ұе…Ҙи§Јжһҗ OpenMatrix зҡ„жһ¶жһ„и®ҫи®ЎпјҢд»ҘеҸҠе®ғеҰӮдҪ•еҖҹйүҙ Harness зҡ„ж ёеҝғжҖқжғіе®һзҺ°еҸҜйқ зҡ„ AI д»»еҠЎзј–жҺ’гҖӮ

## Harness ж ёеҝғжҖқжғіеӣһйЎҫ

Harness жҳҜдёҡз•ҢйўҶе…Ҳзҡ„ DevOps е№іеҸ°пјҢе…¶ж ёеҝғжһ¶жһ„жҖқжғіжңүдёүзӮ№пјҡ

### 1. е§”жүҳжЁЎеһӢпјҲDelegate Architectureпјү

Harness жң¬иә«дёҚжү§иЎҢд»»дҪ•йғЁзҪІж“ҚдҪңпјҢиҖҢжҳҜйҖҡиҝҮ DelegateпјҲд»ЈзҗҶпјүжқҘе®ҢжҲҗпјҡ
- Harness еҸӘеҒҡеҶізӯ–е’Ңзј–жҺ’
- е®һйҷ…жү§иЎҢз”ұ Delegate еңЁзӣ®ж ҮзҺҜеўғдёӯе®ҢжҲҗ
- Delegate еҸҜд»ҘйғЁзҪІеңЁд»»дҪ•ең°ж–№пјҲKubernetesгҖҒVMгҖҒжң¬ең°пјү

### 2. зҠ¶жҖҒжҢҒд№…еҢ–пјҲState Persistenceпјү

Harness е°ҶжүҖжңүжү§иЎҢзҠ¶жҖҒжҢҒд№…еҢ–еӯҳеӮЁпјҡ
- жү§иЎҢеҺҶеҸІгҖҒе®Ўжү№и®°еҪ•гҖҒеҸҳйҮҸзҠ¶жҖҒе…ЁйғЁдҝқеӯҳ
- ж”ҜжҢҒж–ӯзӮ№з»ӯдј пјҢдёӯж–ӯеҗҺеҸҜд»ҘжҒўеӨҚ
- дёҚдҫқиө–еҶ…еӯҳзҠ¶жҖҒпјҢйҮҚеҗҜдёҚдёўеӨұж•°жҚ®

### 3. жөҒзЁӢзј–жҺ’еј•ж“ҺпјҲPipeline Engineпјү

Harness жҸҗдҫӣејәеӨ§зҡ„жөҒзЁӢзј–жҺ’иғҪеҠӣпјҡ
- StageпјҲйҳ¶ж®өпјүзә§еҲ«зҡ„жөҒзЁӢжҺ§еҲ¶
- StepпјҲжӯҘйӘӨпјүзә§еҲ«зҡ„еҺҹеӯҗж“ҚдҪң
- ж”ҜжҢҒе№¶иЎҢжү§иЎҢгҖҒжқЎд»¶еҲҶж”ҜгҖҒе®Ўжү№иҠӮзӮ№
- йҳ»еЎһиҠӮзӮ№дёҚдјҡдёӯж–ӯж•ҙдҪ“жөҒзЁӢ

### AgentRunner зҡ„е§”жүҳжңәеҲ¶

```typescript
// OpenMatrix дёҚзӣҙжҺҘжү§иЎҢпјҢиҖҢжҳҜеҮҶеӨҮ SubagentTask
async prepareSubagentTask(task: Task): Promise<SubagentTask> {
  return {
    subagent_type: this.mapAgentType(task.assignedAgent), // жҳ е°„ Agent зұ»еһӢ
    description: `${task.assignedAgent}: ${task.title}`,
    prompt: await this.buildExecutionPrompt(task), // жһ„е»әе®Ңж•ҙжҸҗзӨәиҜҚ
    isolation: needsIsolation ? 'worktree' : undefined, // жҳҜеҗҰйҡ”зҰ»жү§иЎҢ
    taskId: task.id,
    timeout: task.timeout
  };
}

// Agent зұ»еһӢжҳ е°„
mapAgentType(agentType: AgentType): ClaudeCodeSubagentType {
  const mapping: Record<AgentType, ClaudeCodeSubagentType> = {
    planner: 'Plan',
    coder: 'general-purpose',
    tester: 'general-purpose',
    reviewer: 'general-purpose',
    researcher: 'Explore',
    executor: 'general-purpose'
  };
  return mapping[agentType];
}
```

## еӣӣгҖҒMeeting жңәеҲ¶пјҡйҳ»еЎһдёҚдёӯж–ӯ

Harness жңүдёҖдёӘйҮҚиҰҒзү№жҖ§пјҡе®Ўжү№иҠӮзӮ№дёҚдјҡйҳ»еЎһж•ҙдёӘ PipelineгҖӮOpenMatrix зҡ„ Meeting жңәеҲ¶е®һзҺ°дәҶзұ»дјјж•ҲжһңгҖӮ

### Harness зҡ„е®Ўжү№жңәеҲ¶

```yaml
pipeline:
  stages:
    - name: Build
    - name: Deploy
      approval:
        type: manual
        # зӯүеҫ…е®Ўжү№пјҢдҪҶдёҚйҳ»еЎһеҗҺз»ӯ Stage
```

### OpenMatrix зҡ„ Meeting жңәеҲ¶

```
вқҢ е…¶д»–ж–№жЎҲ:
   TASK-001 вң“ вҶ’ TASK-002 йҳ»еЎһ вҸёпёҸ вҶ’ зӯүз”ЁжҲ·... (жөӘиҙ№ж—¶й—ҙ)

вң… OpenMatrix:
   TASK-001 вң“ вҶ’ TASK-002 йҳ»еЎһ вҶ’ еҲӣе»әMeeting вҶ’ и·іиҝҮ вҶ·
   TASK-003 вң“ вҶ’ TASK-004 вң“ вҶ’ е®ҢжҲҗ!
   вҶ’ з”ЁжҲ·з”Ё /om:meeting з»ҹдёҖеӨ„зҗҶжүҖжңүйҳ»еЎһ
```

### Meeting зҡ„и®ҫи®ЎзҗҶеҝө

**жңҖеӨ§еүҚиҝӣи·қзҰ»пјҲMaximum Forward Progressпјү**пјҡ

- жүҝи®Ө AI жңүдёҚзЎ®е®ҡжҖ§
- дёҚеҒҮиЈ…е®ҢзҫҺпјҢиҖҢжҳҜз”ЁжөҒзЁӢз®ЎзҗҶдёҚзЎ®е®ҡжҖ§
- йҒҮеҲ°йҳ»еЎһ вҶ’ и®°еҪ• вҶ’ и·іиҝҮ вҶ’ з»§з»ӯе…¶д»–д»»еҠЎ
- з”ЁжҲ·еҫ—еҲ°гҖҢеҸҜйў„жңҹзҡ„йғЁеҲҶе®ҢжҲҗгҖҚпјҢиҖҢдёҚжҳҜгҖҢдёҚеҸҜйў„жңҹзҡ„е…ЁйғЁеӨұиҙҘгҖҚ

### дёҺ Harness зҡ„зұ»жҜ”

| Harness зү№жҖ§ | OpenMatrix е®һзҺ° | дјҳеҠҝ |
|-------------|----------------|------|

| е®Ўжү№дёҚйҳ»еЎһ | Meeting жңәеҲ¶ | жңҖеӨ§еүҚиҝӣи·қзҰ»пјҢж•ҲзҺҮжңҖеӨ§еҢ– |

### OpenMatrix зҡ„зӢ¬зү№д»·еҖј

```
еҸҜйқ жҖ§ = е§”жүҳжү§иЎҢпјҲи·ҹйҡҸжЁЎеһӢеҚҮзә§пјү
       + жҢҒд№…еҢ–еҫӘзҺҜпјҲдёҚеҸ—дёҠдёӢж–ҮйҷҗеҲ¶пјү
       + жңҖеӨ§еүҚиҝӣи·қзҰ»пјҲйҳ»еЎһдёҚдёӯж–ӯпјү

д»·еҖј = иҮӘеҠЁеҢ–зЁӢеәҰ Г— иҙЁйҮҸдҝқйҡң / дәәе·Ҙд»Ӣе…Ҙж¬Ўж•°
```

### дёәд»Җд№ҲиҝҷдёӘжһ¶жһ„дјҡжҲҗеҠҹпјҹ

1. **и¶ӢеҠҝеҜ№**пјҡAI зј–з Ғе·Іиҝӣе…ҘгҖҢж•ўдёҚж•ўз”ЁгҖҚйҳ¶ж®ө
2. **еҲҮе…ҘзӮ№еҜ№**пјҡдёҚеҒҡ AIгҖҒдёҚеҒҡ IDEпјҢеҒҡжөҒзЁӢзј–жҺ’еұӮ
3. **жһ¶жһ„еҜ№**пјҡе§”жүҳжЁЎеһӢ + жҢҒд№…еҢ–зҠ¶жҖҒпјҢеҸҜжү©еұ•гҖҒеҸҜжҒўеӨҚ
4. **е“ІеӯҰеҜ№**пјҡдёҚиҝҪжұӮе®ҢзҫҺпјҢиҝҪжұӮжңҖеӨ§еүҚиҝӣи·қзҰ»
