Phase 与 Task

层级结构

Phase 1: 设计阶段
  └─ TASK-001: 架构设计

Phase 2: 开发阶段
  ├─ TASK-002: 数据模型
  ├─ TASK-003: API 接口
  └─ TASK-004: 前端页面

Phase 3: 测试阶段
  ├─ TASK-005: 单元测试
  └─ TASK-006: 集成测试

Phase 阶段

  • Plan - 任务规划
  • TDD - 测试先行 (strict 模式)
  • Develop - 代码实现
  • Verify - 质量门禁
  • Accept - AI 验收

Task 任务

每个 Task 由 Subagent 执行，包含：

  • 明确的输入输出
  • 验收标准
  • 执行日志
