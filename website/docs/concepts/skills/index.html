Skills 命令

OpenMatrix 通过 Claude Code Skills 提供任务编排能力。

命令说明
/om:start启动新任务 (第一个问题选质量级别)
/om:auto全自动执行 - 无阻塞、无确认
/om:status查看当前状态
/om:approve审批决策
/om:meeting处理阻塞问题
/om:resume恢复中断
/om:retry重试失败
/om:report生成报告

安装

npm install -g openmatrix
openmatrix install-skills
ls ~/.claude/commands/om/
