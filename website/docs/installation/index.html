安装

系统要求

  • Node.js >= 18.0.0
  • npm / yarn / pnpm
  • Claude Code CLI

NPM 安装 (推荐)

npm install -g openmatrix

Yarn 安装

yarn global add openmatrix

PNPM 安装

pnpm add -g openmatrix

验证安装

openmatrix --version
openmatrix install-skills
ls ~/.claude/commands/om/

从源码安装

git clone https://github.com/bigfish1913/openmatrix.git
cd openmatrix
npm install
npm run build
npm link
