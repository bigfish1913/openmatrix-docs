Skills 命令

命令说明
/om:start启动新任务 (第一个问题选质量级别)
/om:auto全自动执行 - 无阻塞、无确认
/om:status查看当前状态
/om:approve审批决策
/om:meeting处理阻塞问题
/om:resume恢复中断
/om:retry重试失败
/om:report生成报告

CLI 命令

openmatrix --version          # 查看版本
openmatrix install-skills     # 安装 Skills
openmatrix start --init-only  # 仅初始化目录
