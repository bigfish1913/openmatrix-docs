CLI 命令

Skills 命令

命令说明
/om默认入口 - 直接输入任务描述即可启动
/om:start启动新任务 (第一个问题选质量级别)
/om:auto全自动执行 - 无阻塞、无确认
/om:check自动检测项目改进建议
/om:status查看当前状态
/om:approve审批决策
/om:meeting处理阻塞问题
/om:resume恢复中断
/om:retry重试失败
/om:report生成报告

🪄 自动调用

安装后，直接输入任务描述即可自动调用 /om：

# 无需输入命令，直接描述任务
实现用户登录功能

# 自动调用
/om:start 实现用户登录功能

支持的触发词：

  • 中文：实现...、添加...、修复...、优化...
  • 英文：implement...、add...、fix...、build...

CLI 命令

openmatrix --version          # 查看版本
openmatrix install-skills     # 安装 Skills
openmatrix start --init-only  # 仅初始化目录

start 命令选项

openmatrix start 支持以下选项，用于非交互式启动：

选项说明示例
--init-only仅初始化 .openmatrix 目录openmatrix start --init-only
--title任务标题--title "用户登录"
--description任务描述--description "实现JWT认证"
-q, --quality质量级别 (strict/balanced/fast)-q strict
-t, --tech-stack技术栈-t "React + Node"
--docs文档级别 (full/minimal/none)--docs minimal

非交互式示例

# 直接启动 strict 级别任务，无需交互
openmatrix start --title "用户登录" --description "实现JWT认证" -q strict

# 指定技术栈
openmatrix start --title "API开发" -q balanced -t "Node.js + PostgreSQL"
