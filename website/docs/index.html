快速开始

OpenMatrix 是一个 AI 原生的任务编排框架，帮助你管理复杂的开发项目。 本指南将帮助你在 5 分钟内上手。

安装

OpenMatrix 需要 Node.js 18+ 环境。你可以通过 npm 全局安装：

npm install -g openmatrix

或者使用 npx 直接运行：

npx openmatrix start

第一个任务

在你的项目目录中运行以下命令：

npx openmatrix start "实现用户认证系统"

OpenMatrix 会自动：

  1. 分析任务需求
  2. 拆解为多个 Phase 和 Task
  3. 询问质量级别
  4. 开始执行任务

质量级别

OpenMatrix 提供三种质量级别：

级别 TDD 覆盖率 适用场景
strict ✅ >80% 生产代码
balanced ❌ >60% 日常开发
fast ❌ >20% 快速原型
