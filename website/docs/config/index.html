配置

在项目根目录创建 .openmatrixrc.json：

{
  "quality": {
    "tdd": false,
    "minCoverage": 60,
    "strictLint": true,
    "securityScan": true
  },
  "approvalPoints": ["plan", "merge"],
  "agents": { "maxConcurrent": 3 }
}

配置项

配置类型默认值说明
quality.tddbooleanfalse是否启用 TDD 模式
quality.minCoveragenumber60最低覆盖率要求
quality.strictLintbooleantrue严格 Lint 检查
quality.securityScanbooleantrue安全扫描
approvalPointsstring[][]审批点
